## 타겟 공급사 분석

### 컨택 전 체크리스트
- [ ] 회사 규모 및 연혁 파악
- [ ] 주요 제품군 및 베스트셀러 확인
- [ ] 현재 유통 채널 조사 (온라인/오프라인)
- [ ] 경쟁사 제품 대비 강점 분석
- [ ] 시장 평균 공급가 조사
- [ ] 담당자 연락처 확보

### 유망 공급사 특징
| 구분 | 기준 |
|------|------|
| 제품 | 육아/리빙/식품/뷰티 카테고리 |
| 규모 | 연 매출 1억 이상 ~ 중견기업 |
| 온라인 경험 | 스마트스토어, 쿠팡 등 운영 중 |
| 브랜드력 | 자체 브랜드 보유, SNS 활동 있음 |
| 공급 능력 | MOQ 유연, 안정적 재고 보유 |

### 컨택 우선순위
```
[A등급: 최우선 컨택]
- 우리 플랫폼 타겟과 정확히 맞는 제품
- 이미 SNS 공구 경험 있는 브랜드
- 가격 경쟁력 있는 제품

[B등급: 적극 컨택]
- 제품력은 좋으나 인지도 낮은 브랜드
- 신제품 런칭 예정인 회사
- 온라인 확장 희망하는 오프라인 위주 업체

[C등급: 기회 탐색]
- 대형 브랜드 (경쟁 많음)
- 제품 단가가 높은 프리미엄 브랜드
```

## 전화 통화 스크립트

### 1단계: 본론 진입
```
네, 시간 내주셔서 감사합니다.
간단히 5분 정도만 말씀드릴게요.

저희 모여라딜은 인플루언서 공구 플랫폼인데요,
[공급사명]의 [제품명]이
저희 셀러분들 콘텐츠랑 잘 맞을 것 같아서 연락드렸어요.
```

### 2단계: 모여라딜 소개
```
저희 모여라딜 간단히 소개드리면요,

인스타그램, 블로그에서 활동하는
밸류 엠버서더 1,000명 이상이 함께하는 공구 플랫폼이에요.

제품이 시장에 빠르게 노출되고
실구매로 이어지는 구조를 만들어드려요.

특히 육아, 리빙, 식품 카테고리가 강하고요,
2030 여성분들, 육아맘 고객층이 많아서 구매력이 높아요.
```

### 3단계: 입점 혜택 강조
```
[공급사명] 입장에서 저희랑 협업하시면 좋은 점이요,

첫째, 초기비용 0원이에요.
광고비도 0원, 리스크도 0원이에요.
공급단가만 주시면 끝이고, 팔릴 때만 수수료 차감해요.

둘째, 쿠팡 같은 플랫폼보다 단순해요.
판매수수료만 20~30% 정도고,
광고비, CS, 배송 비용 다 저희가 처리해요.
실수령이 더 많이 남으세요.

셋째, 진짜 후기로 자연 판매돼요.
엠버서더들이 직접 써보고 인스타, 블로그에 올리니까
광고 아닌 진짜 후기로 신뢰가 쌓여요.

넷째, 체험단/페이백 등 소비자 혜택도 제공하고
공구가로 물량 2배 이상 판매도 가능해요.
```

### 4단계: 관심도 확인
```
혹시 입점에 관심 있으세요?

(관심 있다면)
그럼 간단하게 진행해볼 수 있어요.
입점신청은 1분이면 끝나요.
상품 정보랑 공급가만 주시면 돼요.

저희가 확인 후에 제품에 맞는 엠버서더 추천해서
공동구매 연결 도와드릴게요.

(관심 없거나 고민 중이라면)
네, 충분히 검토해보시고요.
혹시 나중에 관심 생기시면 언제든 연락주세요!
```

### 5단계: 다음 단계 제안
```
(관심 있는 경우)

그러면 이렇게 진행해볼게요.

1. 입점신청 링크 보내드릴게요 (1분이면 끝나요)
2. 제품 정보 URL이나 카탈로그 있으시면 전달해주세요
3. 공동구매 판매가나 수수료%, 공급단가 알려주시면
4. 저희가 검토 후 엠버서더 매칭해서 공구 진행해드려요

이메일이나 카톡 알려주시면
입점신청 링크 바로 보내드릴게요.
```

### 6단계: 마무리
```
네, 감사합니다.
[이메일 주소] 맞죠?

그럼 오늘 중으로 메일 보내드릴게요.
검토하시고 궁금한 거 있으시면 편하게 연락주세요.

감사합니다, [담당자명]님.
좋은 하루 되세요!
```

## 미팅 스크립트

### 오프라인/화상 미팅 진행 순서

#### 1. 오프닝 (5분)
```
[담당자명]님, 안녕하세요.
바쁘신데 시간 내주셔서 감사합니다.

오늘 미팅에서는
1. 저희 모여라딜 소개
2. 협업 방식 설명
3. [공급사명] 제품에 맞는 제안
4. 질의응답

이렇게 진행할 예정인데요,
혹시 특별히 궁금하신 부분 있으시면 먼저 말씀해주세요.
```

#### 2. 회사 소개 (10분)
```
먼저 저희 모여라딜에 대해 간단히 소개드릴게요.

[회사 소개서 활용]

저희는 인플루언서 공구 플랫폼으로,
브랜드와 인플루언서를 연결해서
Win-Win 할 수 있는 구조를 만들고 있어요.

📌 주요 지표
- 등록 셀러: [X]명
- 월 평균 공구: [X]건
- 평균 공구 판매량: [X]개
- 주요 카테고리: 육아, 리빙, 식품, 뷰티

📌 특징
- 마이크로 인플루언서 중심
- 실사용 리뷰 기반 판매
- CS/물류 원스톱 서비스
```

#### 3. 협업 방식 설명 (10분)
```
협업 진행 프로세스를 말씀드릴게요.

[1단계: 상품 등록]
- 제품 정보, 이미지 등록
- 공급가 및 판매가 협의
- 재고 확보

[2단계: 셀러 매칭]
- 제품에 맞는 셀러 선정
- 샘플 발송 및 리뷰

[3단계: 공구 진행]
- 셀러 콘텐츠 업로드
- 판매 링크 공유
- 주문 접수 및 배송

[4단계: 정산]
- 공구 종료 후 [X]일 이내 정산
- 판매 리포트 제공

[공급사명]은 상품 등록과 재고 공급만 해주시면
나머지는 저희가 전부 처리합니다.
```

#### 4. 맞춤 제안 (10분)
```
[제품명] 기준으로 제안드리면요,

📌 예상 판매가
- 공급가: [X]원 (협의)
- 판매가: [Y]원
- 셀러 수수료: [Z]%

📌 기대 효과
- 월 [X]건 이상 판매 가능 예상
- [카테고리] 셀러 [X]명 매칭 가능
- 초도 공구 2주 내 진행 가능

📌 초기 조건 제안
- 테스트 공구: [X]개 소량 시작
- 단가: 시장가 기준 [X]% 수준
- 반응 보고 물량 확대

어떻게 생각하세요?
```

#### 5. 질의응답 (10분)
```
제가 설명드린 내용 중에
궁금하신 거나 추가로 논의할 부분 있으실까요?

(질문에 대응 - FAQ 섹션 참고)
```

#### 6. 클로징 (5분)
```
네, 오늘 좋은 이야기 나눠주셔서 감사합니다.

다음 단계로요,
1. 제가 오늘 논의한 내용 정리해서 이메일로 보내드릴게요
2. 검토하시고 진행 의향 알려주시면
3. 샘플 받아서 바로 셀러 매칭 시작하겠습니다

[X]일까지 회신 주시면 감사하겠습니다.

긴 시간 함께해주셔서 감사합니다!
```

## 협업 조건 제안

### 기본 거래 조건
```
📌 공급 조건
- 공급가: 소비자가 대비 [X]% 수준
- MOQ: 초도 [X]개 / 추가 [X]개 단위
- 결제: 정산 시 차감 또는 선결제

📌 판매 조건
- 판매가: 저희 MD와 협의
- 수수료: 없음 (공급가에 마진 포함)
- 프로모션: 셀러 혜택 별도 협의

📌 운영 조건
- 재고: 저희 창고 입고 또는 직송
- 배송: 저희 택배 계약 활용
- CS: 100% 저희가 처리
- 교환/반품: 저희가 1차 대응, 불량은 귀사 처리

📌 정산 조건
- 정산 주기: 공구 종료 후 [X]일
- 정산 방식: 세금계산서 발행 후 입금
```

### MOQ 협상 가이드
| 단계 | MOQ | 공급가 할인 | 비고 |
|------|-----|------------|------|

## 자주 받는 질문 대응

### Q1: "수수료는 어떻게 되나요?"
```
저희는 별도 수수료를 받지 않아요.
공급가에 마진을 반영해서 판매가를 책정하는 방식이에요.

예를 들어,
공급가 10,000원 → 판매가 15,000~18,000원
이런 식으로 진행됩니다.

마진율은 보통 30~50% 수준으로 잡고요,
제품 특성이나 경쟁 상황에 따라 조율합니다.
```

### Q2: "최소 물량이 얼마예요?"
```
첫 공구는 테스트 성격이라
30~50개 정도 소량으로 시작하실 수 있어요.

반응 좋으면 바로 추가 발주 들어가고,
정기적으로 진행하시면 100개, 300개 단위로 늘어나요.

초기 리스크 줄이면서 시작하시면 됩니다.
```

### Q3: "재고는 어디에 보관하나요?"
```
두 가지 방식이 있어요.

[방식 1: 위탁 입고]
저희 창고에 입고해두시면
주문 들어올 때마다 바로 출고해요.
재고 관리 부담 없으시죠.

[방식 2: 직송]
주문 데이터 전달드리면
직접 출고해주시는 방식이에요.
재고 따로 넘기시기 어려울 때 가능해요.

어떤 방식이 편하세요?
```

### Q4: "정산은 언제 해주시나요?"
```
공구 종료 후 [X]일 이내에 정산드려요.

정산일에 세금계산서 발행하시면
확인 후 바로 입금해드리고 있어요.

매 공구마다 판매 리포트도 같이 드리니까
투명하게 확인하실 수 있어요.
```

### Q5: "CS는 어떻게 되나요?"
```
CS는 100% 저희가 처리해요.

일반 문의, 배송 문의, 교환/반품까지
전부 저희 CS팀에서 대응하고요,

제품 자체 불량이나 품질 이슈만
귀사로 연락드리고 있어요.

[공급사명]은 제품만 잘 공급해주시면 됩니다.
```

### Q6: "다른 채널이랑 가격 충돌 나면 어떡하죠?"
```
저희도 그 부분 중요하게 생각해요.

보통 두 가지 방식으로 조율해요.

[방식 1: 가격 맞춤]
기존 채널 가격에 맞춰서 판매가 책정해요.
같은 가격인데 인플루언서 노출 효과를 얻는 거죠.

[방식 2: 번들/구성 차별화]
같은 제품이라도 구성을 다르게 해서
직접 비교가 안 되게 하는 방법도 있어요.

어떤 방식이 [공급사명]에 맞으실까요?
```

### Q7: "효과가 있을까요? 실적 있어요?"
```
저희랑 비슷한 [카테고리] 제품으로
[사례 브랜드명]이 [X]개월 동안 [Y]개 팔았어요.

특히 인플루언서 리뷰 콘텐츠 덕분에
브랜드 인지도도 많이 올라갔다고 하시더라구요.

첫 공구는 테스트로 작게 시작하시고,
반응 보고 확대하시면 리스크 적어요.
```

### Q8: "대형 브랜드도 하세요?"
```
네, [예시 브랜드]도 저희랑 협업 중이에요.

대형 브랜드는 보통
신제품 런칭이나 특정 타겟 공략할 때 활용하시더라구요.

인플루언서 팬층이 명확하니까
타겟 마케팅 효과가 좋아요.
```
