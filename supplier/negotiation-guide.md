## 초기 컨택 스크립트

### 1단계: 인사 및 소개
```
안녕하세요, [회사명]의 [담당자명]입니다.

저희는 [플랫폼 특성]을 중심으로 운영되는 이커머스 플랫폼입니다.
귀사의 [제품명/제품군]에 관심이 있어 연락드렸습니다.

잠깐 시간 내주실 수 있으실까요?
```

### 2단계: 플랫폼 소개
```
저희 플랫폼은 현재 다음과 같은 특징이 있습니다:

1. 월 방문자 수: [X]명
2. 주요 고객층: [타겟 고객 설명]
3. 평균 전환율: [X]%
4. 주요 카테고리: [카테고리명]

귀사의 제품이 저희 플랫폼에 잘 맞을 것으로 판단됩니다.
```

### 3단계: 협력 제안
```
저희와 함께 다음과 같은 방식으로 협력하면 어떨까 합니다:

1. 입점 수수료: [X]% (업계 평균 대비 경쟁력 있는 수준)
2. 마케팅 지원: [구체적인 지원 내용]
3. 정산 주기: [주기]
4. 물류 지원: [지원 사항]

귀사는 어떻게 생각하시나요?
```

## MOQ 및 단가 협상 스크립트

### 1단계: 기본 단가 확인
```
먼저 기본적인 거래 조건을 확인하고 싶습니다.

Q1: 현재 [제품명]의 공급단가는 어떻게 되시나요?
Q2: 최소 주문 수량(MOQ)이 있으신가요?
Q3: 대량 구매 시 할인 정책이 있으신가요?
```

### 2단계: MOQ 연동 단가 협상
```
저희가 제안드리고 싶은 것은 단계별 단가 구조입니다.

[옵션 A: 초기 진입 단가]
- 초기 런칭 기간 (1-3개월)
- MOQ: [X]개
- 공급단가: [Y]원
- 목적: 시장 테스트 및 판매 데이터 확보

[옵션 B: 일반 거래 단가]
- MOQ 월 [X]개 달성 시
- 공급단가: [Y-A]원 (기존 대비 [X]% 인하)
- 추가 혜택: [마케팅 지원 등]

[옵션 C: 대량 거래 단가]
- MOQ 월 [Z]개 달성 시
- 공급단가: [Y-B]원 (기존 대비 [X]% 인하)
- 추가 혜택: [전용 페이지 제작, 메인 배너 노출 등]

이렇게 하면 초기에는 서로 리스크를 줄이고,
판매가 늘어날수록 서로 이익이 커지는 구조입니다.

어떻게 생각하시나요?
```

### 3단계: 구체적인 숫자 협상
```
[공급사 응답 후]

네, 말씀하신 부분 충분히 이해됩니다.
그럼 이렇게 조율해보면 어떨까요?

1단계 (테스트 기간 3개월)
- MOQ: 월 [X]개
- 단가: [Y]원
- 결제 조건: [조건]

2단계 (MOQ 달성 시)
- MOQ: 월 [X*2]개 달성 시
- 단가: [Y*0.95]원 (5% 인하)
- 재고 관리: [방식]

3단계 (대량 주문 시)
- MOQ: 월 [X*5]개 달성 시
- 단가: [Y*0.90]원 (10% 인하)
- 독점 혜택: [혜택 내용]

이 구조가 현실적으로 가능하실까요?
```

### 4단계: 조건 조율
```
[가격이 부담스러울 경우]

가격 부분에 대해서는 이해합니다.
그럼 다음과 같은 방식은 어떨까요?

옵션 1: 단계적 인하
- 1-3개월: 현재 단가 유지
- 4-6개월: MOQ [X]개 달성 시 [Y]% 인하
- 7개월~: MOQ [Z]개 달성 시 [Z]% 인하

옵션 2: 성과 기반 인센티브
- 기본 단가는 유지하되
- 분기별 목표 달성 시 리베이트 지급
- 목표: 분기 [X]개 판매 → [Y]% 리베이트

옵션 3: 프로모션 분담
- 단가는 현재 제안 수준으로 진행
- 대신 프로모션 비용을 [X]:[Y] 비율로 분담
- 판매 증대 시 서로 이익

이 중에서 귀사에 더 맞는 방식이 있으실까요?
```

### 5단계: 부가 조건 협상
```
단가 외에 몇 가지 더 논의하고 싶은 부분이 있습니다:

1. 결제 조건
   - 현재: [현재 조건]
   - 제안: [제안 조건]
   - 이유: [이유]

2. 재고 관리
   - 저희 창고 입고 vs 직송
   - 안전 재고 수량
   - 발주 리드타임

3. 반품/교환 정책
   - 불량률 기준
   - 반품 처리 프로세스
   - 비용 부담 주체

4. 계약 기간
   - 초기 계약: [X]개월
   - 자동 갱신 조건
   - 해지 조건

이 부분들도 함께 정리하면 좋을 것 같은데, 어떠신가요?
```

## 협상 후 후속 조치

### 1. 즉시 조치 (협상 당일)
```
오늘 좋은 대화 나눠주셔서 감사합니다.

오늘 합의된 내용을 정리하면:
1. [합의 사항 1]
2. [합의 사항 2]
3. [합의 사항 3]

[X]일까지 계약서 초안을 보내드리겠습니다.
검토 후 의견 주시면 최종 조율하겠습니다.
```

### 2. 계약서 발송 (1-2일 내)
```
안녕하세요, [담당자명]입니다.

지난 협상 내용을 바탕으로 계약서를 작성했습니다.
첨부 파일 확인 부탁드립니다.

특히 다음 부분 중점 검토 부탁드립니다:
- 3조: 가격 조건 (MOQ 연동 단가)
- 5조: 결제 조건
- 7조: 계약 기간 및 갱신

[X]일까지 검토 의견 주시면 감사하겠습니다.
```

### 3. 킥오프 미팅 (계약 후 1주일 내)
```
계약 체결을 축하드립니다!

본격적인 협업을 위해 킥오프 미팅을 제안드립니다.

안건:
1. 상품 등록 프로세스 안내
2. 시스템 사용법 교육
3. 초기 입고 일정 협의
4. 마케팅 계획 공유
5. 정산 프로세스 안내

가능하신 날짜: [날짜 옵션 제시]
```

## 자주 묻는 질문 (FAQ)

### Q1: 공급사가 단가 인하를 거부하면?
**A**: 다른 혜택으로 보상
- 프로모션 비용 분담
- 마케팅 지원 확대
- 결제 조건 개선 (선결제 등)
- 장기 계약으로 안정성 보장

### Q2: MOQ를 달성하지 못하면?
**A**: 단계적 접근
1. 분기별로 재평가
2. 목표 조정 협의
3. 초기 3개월은 테스트 기간으로 설정
4. 달성률에 따라 유연하게 조정

### Q3: 계약 기간은 어떻게?
**A**:
- 초기 계약: 6개월-1년
- 성과 확인 후 장기 계약 전환
- 자동 갱신 조항 포함
- 3개월 전 해지 통보 조항

### Q4: 경쟁사보다 단가가 높으면?
**A**: 차별화 포인트 강조
- 플랫폼 트래픽 품질
- 타겟 고객 정확도
- 마케팅 지원
- 정산 신속성
- 성장 가능성
