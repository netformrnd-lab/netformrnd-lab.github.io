## 타겟 셀러 분석

### 컨택 전 체크리스트
- [ ] 팔로워/구독자 수 확인 (1,000명 이상 권장)
- [ ] 최근 게시물 반응률 체크 (좋아요, 댓글)
- [ ] 콘텐츠 카테고리 확인 (육아, 리빙, 식품, 뷰티 등)
- [ ] 기존 공구/협찬 경험 여부 파악
- [ ] 팔로워 특성 분석 (연령대, 관심사)

### 이상적인 셀러 프로필
| 구분 | 기준 |
|------|------|
| 팔로워 | 1,000명 이상 (마이크로: 1,000~10,000) |
| 반응률 | 게시물당 좋아요 50개 이상, 댓글 10개 이상 |
| 콘텐츠 | 제품 리뷰, 일상 공유, 정보성 콘텐츠 |
| 활동빈도 | 주 3회 이상 포스팅 |
| 톤앤매너 | 친근하고 신뢰감 있는 이미지 |

## 협업 조건 제안

### 기본 협업 구조
```
[셀러명]님께 제안드리는 협업 조건입니다.

📌 기본 조건
- 역할: 콘텐츠 제작 + 판매 링크 공유
- 수수료: 판매가의 [X]%
- 정산: 공구 종료 후 [X]일 이내

📌 저희가 지원하는 것
- 샘플 무료 제공
- 상세페이지 이미지 제공
- 주문/배송/CS 전담 처리
- 전담 MD 배정

📌 셀러님이 하실 것
- 제품 리뷰 콘텐츠 1건 이상
- 스토리/피드에 판매 링크 공유
- 팔로워 질문 응대 (기본적인 것만)
```

### 수수료 구조
| 구분 | 팔로워 기준 | 기본 수수료 | 성과 보너스 |
|------|------------|------------|------------|
| 신규 셀러 | 1,000~5,000 | [X]% | - |
| 일반 셀러 | 5,000~10,000 | [X]% | +[Y]% (월 [Z]건 이상) |
| 우수 셀러 | 10,000 이상 | [X]% | +[Y]% (월 [Z]건 이상) |

### 단계별 혜택
```
[1단계: 신규 셀러]
- 기본 수수료: [X]%
- 샘플 제공: 공구당 1개
- 지원: 기본 이미지 제공
```

## 자주 받는 질문 대응

### Q1: "수수료가 얼마예요?"
```
기본적으로 판매가의 [X]%를 드리고 있어요.
다른 플랫폼보다 높은 편이에요.

그리고 공구 많이 하시면 등급이 올라가면서
수수료도 더 높아져요.

VIP 셀러분들은 [X+5]%까지 받고 계세요.
```

### Q2: "제가 직접 배송해야 하나요?"
```
아니요, 전혀요!

[셀러명]님은 콘텐츠 만들고 링크 공유만 하시면 돼요.
주문 확인, 포장, 배송, CS까지 전부 저희가 처리합니다.

문의 들어오면 저희 CS팀에서 바로 응대하고,
[셀러명]님한테는 간단한 것만 공유드려요.
```

### Q3: "팔로워가 적은데 괜찮을까요?"
```
물론이죠!

저희는 팔로워 수보다 반응률을 더 중요하게 봐요.
[셀러명]님처럼 팔로워분들과 소통 잘 하시는 분이
실제 판매로도 잘 이어지거든요.

처음에는 작게 시작하시고,
반응 보면서 점점 늘려가시면 돼요.
```

### Q4: "다른 플랫폼에서도 공구하고 있는데요"
```
전혀 상관없어요!

오히려 경험 있으신 분이라 더 좋아요.
다른 플랫폼이랑 비교해보시고,
저희가 더 좋은 조건이라고 느끼시면 더 자주 해주시면 되고요.

중복으로 활동하셔도 전혀 문제없습니다.
```
