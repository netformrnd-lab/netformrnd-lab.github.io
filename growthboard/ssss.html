01

Why 모여라딜?

기존 플랫폼의 한계를 해결하는 신뢰 기반 공동구매

🏠 기존 플랫폼

  • ✕판매상품 보완 제한 (상페, 썸네일, 옵션 등)
  • ✕콘텐츠는 포스팅형 위주
  • ✕MD와 조율은 판매가/할인가만
  • ✕상시판매 → 마진율 측정 어려움
  • ✕사은품 지원 불가

◈ 모여라딜

  • ✓상페, 썸네일, 옵션 부가서비스 제공
  • ✓사은품 지원 형태 가능
  • ✓대량판매 / 기간·물량 한정
  • ✓정확한 1개당 마진율 측정
  • ✓자체 양성 셀러 보유

↓

🔗 데이터 연결 흐름 설명

위 예시는 '입점신청 → 정산'까지의 흐름 중 하나입니다.

모여라딜의 빅데이터 시스템은 모든 사이트와 센터가 Firebase를 중심으로 실시간 연동됩니다.

  • 파트너포털 → 공급사/셀러 가입신청 → Firebase → 관리센터에서 확인/승인

  • 셀러사이트 → 판매 신청/정산조회 → Firebase → CRM센터에서 관리
  • 모든 데이터 → 전략센터에서 KPI 분석 및 인사이트 도출

행성들을 클릭하면 각 센터/사이트의 세부 기능을 확인할 수 있습니다.
