# 모여라딜 (Moyeora Deal)

공동구매 딜 관리 시스템

## 페이지 구성

| 페이지 | 설명 | URL |
|--------|------|-----|
| 관리자 대시보드 | 공구/공급사/셀러 관리 | `moyeora-deal-manager.html` |
| 파트너 로그인 | 공급사/셀러 로그인 | `moyeora-login.html` |
| 파트너 캘린더 | 공급사/셀러 일정 조회 | `moyeora-calendar-public.html` |

## 기능

### 관리자
- 📅 캘린더: 공구 일정 한눈에 확인
- 📦 공구 목록: 전체 공구 관리
- 🏭 공급사: 공급사 정보 및 계약 관리
- 📦 공급사 상품 DB: 상품 정보 관리
- 👤 셀러: 셀러 정보 관리
- 📋 셀러 리스트: 셀러별 공구/매출 현황
- 🚀 공구 진행 관리: 샘플/결제링크/컨텐츠 전달 관리
- 📊 매출관리 내역서: 정산 및 매출 관리

### 파트너
- 📅 내 공구 일정 확인
- 📋 공구별 상세 정보 조회
- ✅ 진행 상황 확인

## 기술 스택

- HTML/CSS/JavaScript (Vanilla)
- Firebase Firestore (데이터베이스)
- GitHub Pages (호스팅)
